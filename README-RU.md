## Russian README
### 🖥 Обычный режим (без `#`)
Откройте страницу как обычно, чтобы использовать интерактивный интерфейс. Выберите длину пароля, включите нужные наборы символов (строчные, заглавные, цифры, спецсимволы, безопасный режим), переключите светлую/тёмную тему, сгенерируйте и скопируйте результат в один клик. Идеально для повседневного использования.

### 🔗 Режим URL-хэша (с `#`)
Добавьте параметры прямо в адресную строку, чтобы пропустить интерфейс и мгновенно получить результат. Создан для закладок, скриптов, автоматизации или встраивания. Возвращает обычный текст или JSON.

### ⚙️ Как настроить хэш
Формат: `#<длина>-<наборы>[,<режим>]`
- **Наборы:** `az` (строчные), `AZ` (заглавные), `09` (цифры), `sym` (символы), `safe` (исключить `i/l/1/O/0`)
- **Режимы:** `json` (вернёт метаданные), `plain` (вернёт только пароль)
- **Примеры:**
  - `#16-az,AZ,09,sym` → Интерфейс со стандартным паролем
  - `#32-az,09,json` → Вывод в JSON
  - `#64-az,AZ,09,sym,safe,plain` → Только текст, безопасные символы

### 🛡️ Почему это безопасно и приватно
- ✅ **Полностью клиентская генерация:** Всё происходит в вашем браузере. Никаких сетевых запросов, аналитики и отслеживания.
- ✅ **Криптографическая стойкость:** Используется `crypto.getRandomValues()` с алгоритмом отсечения для устранения смещения (modulo bias).
- ✅ **Нулевое хранение:** Пароли не сохраняются, не кэшируются и не передаются. Закрытие вкладки стирает всё.
- ✅ **Работает оффлайн:** Не требует интернета. Открытый исходный код.
